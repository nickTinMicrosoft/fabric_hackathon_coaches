# DataFiles

This folder contains all raw data assets used in the hackathon challenges. Participants will ingest these files into Microsoft Fabric for analysis and reporting.

---

## Folders

| Folder | Description |
|--------|-------------|
| [parquet](parquet/) | Parquet data files and supporting image assets |

---

## Usage

The data in this folder is ready to be uploaded to a Fabric Lakehouse or used directly by scripts in the [Scripts](../Scripts/) folder.
