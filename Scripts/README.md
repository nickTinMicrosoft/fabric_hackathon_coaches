# Scripts

This folder contains all scripts, seed data, and streaming simulators used to provision databases and generate live data for the hackathon. Scripts are organized by target technology.

---

## Folders

| Folder | Description |
|--------|-------------|
| [Cosmos](Cosmos/) | Seed data for Azure Cosmos DB |
| [Python](Python/) | Live streaming simulators and data-generation utilities |
| [SQL](SQL/) | SQL DDL and seed scripts for relational databases |

---

## Quick Reference

| Technology | What It Does |
|------------|-------------|
| **Cosmos** | Provides JSON documents to seed a Cosmos DB container with student grade records |
| **Python** | Streaming simulators (Hospital, Metra trains, Flight Tracker), multi-agent orchestrator, and Parquet file generation |
| **SQL** | Creates tables (Students, Classes, Enrollments) and seeds them with sample data |
