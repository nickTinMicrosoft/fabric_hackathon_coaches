# Scripts

This folder contains all scripts and seed data used to provision and populate databases for the hackathon. Scripts are organized by target technology.

---

## Folders

| Folder | Description |
|--------|-------------|
| [Cosmos](Cosmos/) | Seed data for Azure Cosmos DB |
| [Python](Python/) | Python utilities for generating data files |
| [SQL](SQL/) | SQL DDL and seed scripts for relational databases |

---

## Quick Reference

| Technology | What It Does |
|------------|-------------|
| **Cosmos** | Provides JSON documents to seed a Cosmos DB container with student grade records |
| **Python** | Generates Parquet files from image folders with synthetic medical metadata |
| **SQL** | Creates tables (Students, Classes, Enrollments) and seeds them with sample data |
