# Cosmos DB Scripts

This folder contains seed data and scripts for Azure Cosmos DB used in the hackathon.

---

## Folders

| Folder | Description |
|--------|-------------|
| [Student Grades](Student%20Grades/) | JSON seed data for the Student Grades Cosmos DB container |

---

## Usage

Import the JSON seed files into your Azure Cosmos DB container. The data contains student grade records keyed by `Student_ID` and partitioned by `Class_Room`.
