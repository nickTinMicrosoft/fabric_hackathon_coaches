# Step 1 — Create the KQL Database in Fabric

## Create the KQL Database

1. Open your **Microsoft Fabric workspace**.
2. Click **+ New item** → select **Eventhouse**.
3. Name it: `HospitalEventhouse`
4. Once created, the Eventhouse automatically includes a KQL Database with the same name. You can rename it to `HospitalOpsDB` if desired.

## Verify

- Open the KQL Database and confirm you see the **Query** tab.
- You should see an empty database with no tables yet.
